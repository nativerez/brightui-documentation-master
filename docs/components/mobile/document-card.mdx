# Document card

Displays document details and associated actions.

### Structure

#### Document Object

- **Icon** (indicates file type)
- **Filename** (displays the uploaded file name in the format 'filename.ext')
- **Context menu icon** (actions available, subject to role permissions - such as Details, Rename, Move, Delete)
- **File type and size**
- **Created date**
- **Read receipt required** (icon and text)
- **Acceptance required** (icon and text)
- **Footer** (displays follow-up status)

#### Document Folder

- **Icon** (folder icon)
- **Filename** (displays the folder name)
- **Created date** (displays "Created on DD Mmm YYYY") - shows on user created folders only

### Guidelines

#### When to use

- Within document folders

#### When not to use

- For anything other than a document

#### Properties

- **Show Follow up** (true/false)
- **Body - Requiremnts** (true/false)
- **Requirements - Type** (Read receipt, Acceptance, Both)
- **Requirements - State** (Complete, Incomplete)
- **Follow up - Follow up period** (within 7 days, within 30 days, overdue) Displays as 'Due in X day(s)'

- **File name** - Primary bold

- **Created date** - Secondary regular
- **Requirements** - Secondary regular
- **Follow up status** - Tertiary regular/bold

### Behaviour

Tap kebab menu to display context menu with available actions.
Tap card body to view full screen details with 'Download' floating action button.
