### Guidelines

For guidelines on when to use and when not to use, see the [Loading states](../../design-patterns/loading-states) design pattern

#### Spinner vs. XingLoader

Prefer the spinner for sub-page / component level indeterminate progress. For a full page inderderminate progress indicator, use the [Xingloader](./xingloader).
