### Guidelines

For guidelines on when to use and when not to use, see the [Loading states](../../design-patterns/loading-states) design pattern

#### Spinner vs. XingLoader

Prefer the Xing loader for page-level indeterminate progress. For a sub-page / component-level inderderminate progress indicator, use the [Spinner](./spinner).
